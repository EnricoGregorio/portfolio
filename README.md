# 🚀 Enrico Gregorio | Portfólio Pessoal

Bem-vindo ao repositório do meu portfólio! Este projeto foi desenvolvido para apresentar minha trajetória, habilidades e os sistemas que construo como **Desenvolvedor Back-end**. 

O foco deste desenvolvimento foi aplicar conceitos de arquitetura limpa e escalabilidade em uma interface front-end rápida, responsiva e moderna.

## 🛠️ Tecnologias Utilizadas

Embora meu foco principal seja a construção de sistemas robustos e APIs eficientes utilizando **Java**, **Python** e **TypeScript**, este portfólio foi construído com ferramentas de ponta do ecossistema front-end para garantir a melhor performance:

- **[Next.js](https://nextjs.org/)** (App Router) - Framework React com renderização híbrida (SSR/SSG).
- **[React](https://react.dev/)** - Biblioteca JavaScript para construção de interfaces de usuário.
- **[Tailwind CSS](https://tailwindcss.com/)** - Framework CSS utilitário para estilização rápida e responsiva.
- **[TypeScript](https://www.typescriptlang.org/)** - Superconjunto de JavaScript que adiciona tipagem estática.
- **[EmailJS](https://www.emailjs.com/)** - Serviço de automação para envio de e-mails diretamente pelo lado do cliente.

## ✨ Funcionalidades em Destaque

- **Design Mobile-First:** Interface totalmente responsiva que se adapta perfeitamente a smartphones, tablets e desktops (testada rigorosamente em resoluções como a do iPhone SE).
- **Otimização de Assets:** Uso do componente nativo `<Image>` do Next.js para conversão de formatos modernos (WebP) e lazy-loading de capturas de tela dos meus projetos.
- **Formulário de Contato Protegido:** Integração com EmailJS contendo validação de campos e restrição de domínio (Allowlist) para evitar abusos de requisições.
- **Barra de Rolagem Customizada:** Estilização via WebKit para uma experiência visual alinhada à paleta de cores primárias da marca pessoal.
- **Página de Erro (404) Customizada:** Controle de rotas inexistentes para reter o usuário no ambiente do portfólio.

## ⚙️ Como executar este projeto localmente

Se você deseja rodar uma cópia deste portfólio na sua máquina, siga os passos abaixo:

### Pré-requisitos
- Node.js (v18.17.0 ou superior)
- npm (Gerenciador de pacotes)

### Passo a passo

1. Clone o repositório:
```bash
git clone [https://github.com/EnricoGregorio/portfolio.git](https://github.com/EnricoGregorio/portfolio.git)
```

2. Entre no diretório do projeto:
```bash
cd portfolio
```

3. Instale as dependências
```bash
npm install
```

4. Inicie o servidor de desenvolvimento
```bash
npm run dev
```

5. Abra o seu navegador e acesse [http://localhost:3000](http://localhost:3000)

> Nota: Este projeto substitui a versão 1.0 do meu portfólio (criada em HTML, CSS e JS puros), refletindo minha evolução técnica e adoção de frameworks modernos e componentização.
